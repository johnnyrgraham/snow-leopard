# Snow Leopard — Educational Website

A multi-page educational website about snow leopards, built in 2020 as part of a Certificate in Information Technology at [Your Institution].

This was one of several projects completed during that certificate, demonstrating the ability to research, design, and build a fully structured website from scratch — without AI coding tools.

---

## About the project

The brief was to build a multi-page informational website on a chosen subject. I chose snow leopards.

The site covers:
- **Introduction** — overview and classification
- **Adaptations** — physical traits that allow survival in high-altitude environments
- **Habitat** — range across Central and South Asia
- **Feeding** — prey species and hunting behaviour
- **Behavior** — territorial and social patterns
- **Reproduction** — breeding cycles and cub development
- **References** — cited sources

---

## What I built

- Full multi-page site with consistent navigation across all pages
- A sticky scrolling toolbar — the nav follows as you scroll, inspired by the design of the Need Essentials website
- Responsive image layouts with full-bleed hero photography
- Clean typographic hierarchy across content-heavy pages

The scrolling toolbar was built by adapting and integrating elements from various Stack Overflow and Stack Exchange contributors, modified to fit the site's design. This was pre-AI coding — figuring out how other developers solved problems, reading their code, understanding it, and tailoring it to work in a new context. That process is a core developer skill.

---
